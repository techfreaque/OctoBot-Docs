# current live time

## get current live time

{% hint style="success" %}
**in live mode:**&#x20;

returns the actual live time
{% endhint %}

{% hint style="success" %}
**while backtesting:**

returns the current close time of the lowest timeframe activated
{% endhint %}

```
current_live_time(ctx)
```
